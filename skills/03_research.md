# Skill: Research & Analysis

## When to use
Any request to research, find, summarize, compare, or analyze information from the web or documents.

## Approach
1. Use Tavily for fast web search on current topics
2. Use Firecrawl to scrape and read specific URLs in depth
3. Use browser tool for interactive pages, dashboards, or login-required content
4. Synthesize findings — never just dump raw search results
5. Always cite sources with URLs

## Research Workflow
1. Search broadly first (2-3 Tavily queries)
2. Deep-read the 2-3 most relevant sources with Firecrawl
3. Cross-reference and identify consensus vs conflicting info
4. Produce a structured summary with key insights

## Output Format
- Lead with the answer/conclusion, not the research process
- Use bullet points for findings, numbered list for steps/rankings
- Include source URLs at the bottom
- Flag if information might be outdated

## Oui's Research Context

## Examples
- "What are the latest AI agents frameworks in 2025?"
- "Research competitors to Agoda in Southeast Asia"
- "Find the best open-source alternatives to X"
- "Summarize the key findings from this URL: ..."
